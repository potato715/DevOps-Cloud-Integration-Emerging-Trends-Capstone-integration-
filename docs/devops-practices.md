# DevOps Practices

## Version Control
- Using Git with GitHub
- All changes committed to the main branch

## CI/CD Pipeline
- GitHub Actions runs automatically on every push
- GitHub Pages deploys automatically on every push

## Deployment
- Platform: GitHub Pages (free)
- Live URL: https://potato715.github.io/DevOps-Cloud-Integration-Emerging-Trends-Capstone-integration-

## How to Deploy
1. Make changes to any file in the repository
2. Commit the changes
3. GitHub Pages automatically updates within 1-2 minutes
